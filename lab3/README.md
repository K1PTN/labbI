# 🎮 Platformer Game

Веб-игра в жанре платформер, разработанная на чистом JavaScript с использованием HTML5 Canvas API.

## 🤖 Использование нейросети

| Параметр | Значение |
|----------|----------|
| **Нейросеть** | DeepSeek / ChatGPT |
| **Время выполнения** | ~4.5 часа |
| **Использование ИИ** | Генерация архитектуры, отладка коллизий, оптимизация игрового цикла, создание промптов для классов |

### Детали использования:
- ✅ Структура проекта и разделение на модули
- ✅ Реализация AABB-коллизий с обработкой краёв
- ✅ Логика игрового цикла с фиксированным timestep
- ✅ Обработка ввода с предотвращением залипания клавиш
- ✅ Оптимизация рендеринга и анимаций

## 🎯 Функционал

### Основные механики:
- 🕹️ Управление: ← → / A D для движения, Space / W / ↑ для прыжка
- 🌍 Физика: гравитация, трение, коллизии с платформами
- 🪙 Сбор монет для получения очков
- 👾 Враги с паттерном движения (уровни 2-3)
- ❤️ Система здоровья и условий поражения
- 🏆 3 уровня с возрастающей сложностью

### Дополнительные возможности:
- 💾 Сохранение рекорда и максимального уровня в localStorage
- 🎨 Адаптивный визуальный стиль (день/вечер/ночь)
- ✨ Анимации персонажа, частицы при прыжке
- ⏸️ Система паузы и перезапуска
